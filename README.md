# ☁️ Cloud Computing & Azure Data Fundamentals Project  
*A Data Technician Bootcamp Project*

This project demonstrates my ability to work with **Microsoft Azure** and apply cloud‑based data services to analyse and extract insights from **retail and sales datasets**. It highlights foundational cloud concepts, Azure data tools, and practical skills in building secure, scalable data workflows.

---

## 🚀 Project Overview
The goal of this project was to move from traditional, local data analysis to a **cloud‑enabled analytics environment**. Using Azure services, I ingested, stored, transformed, and queried retail and sales data to uncover trends in product performance, customer behaviour, and revenue patterns.

---

## 🔧 Skills & Technologies Demonstrated

### ☁️ Cloud Computing Fundamentals
- Understanding cloud models: **IaaS, PaaS, SaaS**
- Benefits of cloud adoption: scalability, elasticity, high availability
- Shared responsibility model and cloud governance principles

### 🗄️ Azure Data Services
- **Azure Storage Accounts** — storing structured and semi‑structured retail datasets  
- **Azure Data Lake Gen2** — organising data into hierarchical zones for analysis  
- **Azure SQL Database** — querying data using cloud‑hosted SQL  
- **Azure Synapse Analytics (intro)** — understanding pipelines, workspaces, and analytics capabilities  
- **Azure Data Factory (intro)** — concepts of pipelines, linked services, and data movement  

### 🧹 Data Transformation & Preparation
- Uploading and organising raw CSV files in Azure Storage  
- Structuring data into **Bronze / Silver / Gold** layers  
- Cleaning and preparing data for analysis using SQL in Azure SQL Database  

### 📊 Analytical Skills
Using Azure SQL, I analysed retail and sales data to extract insights such as:

- Top‑selling products and categories  
- Monthly and seasonal revenue trends  
- Store‑level performance comparisons  
- Customer purchasing patterns  
- Identifying underperforming products  

---

## 🧠 What This Project Demonstrates
- Ability to work with **cloud‑based data platforms**  
- Understanding of Azure’s core data services and architecture  
- Skill in building end‑to‑end data workflows in the cloud  
- Confidence querying cloud‑hosted datasets using SQL  
- Practical experience applying cloud analytics to real retail data
